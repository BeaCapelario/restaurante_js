Este repositório contém o desenvolvimento de uma aplicação para um restaurante utilizando JavaScript puro (Vanilla JS). O foco do projeto foi a manipulação dinâmica do DOM para criar uma experiência de usuário fluida, simulando o funcionamento de um cardápio e pedidos online.

---

### O que foi explorado:
* Manipulação do DOM: Alteração dinâmica de elementos HTML via JavaScript.
* Event Listeners: Captura de interações do usuário (cliques, envios, filtros).
* Arrays e Objetos: Estruturação dos dados do cardápio e gerenciamento de itens.
* Lógica de Negócio: Cálculo de subtotais, taxas e validação de pedidos.
* Template Literals: Renderização dinâmica de componentes na interface.

---

### Tecnologias Utilizadas
| Tecnologia | Descrição |
| :--- | :--- |
| JavaScript | Lógica de programação e dinamismo da página. |
| HTML5 | Estruturação dos elementos do restaurante. |
| CSS3 | Estilização do cardápio e interface de usuário. |
